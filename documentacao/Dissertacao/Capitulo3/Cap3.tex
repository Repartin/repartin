\section{Tecnologias e Ferramentas}
O desenvolvimento do Repartin envolve tecnologias modernas tanto para Web quanto 
para mobile, para isso, foi selecionado as seguintes tecnologias: 
\begin{itemize}  
	\item React
	\item React Native
	\item Firebase-Auth
	\item Node
	\item Redux
\end{itemize}

\noindent Também será utilizado as seguintes IDE's para edição e codificação:
\begin{itemize}  
	\item TexStudio - Edição do Documento
	\item Visual Studio Code
	\item Android Studio
\end{itemize}

\noindent E todo versionamento do projeto será feito usando GIT e armazenando em um
repositório do Github.
